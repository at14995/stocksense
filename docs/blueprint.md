# **App Name**: Stock Sense

## Core Features:

- Landing Page: Display a brief overview of the app's features and benefits.
- Dashboard: Provide a personalized dashboard with key stock market data and user-specific information.
- Marketplace: List different products
- Sentiment Analysis: Analyze sentiment related to selected stocks using AI and LLMs to determine general feeling and recommend trading accordingly. Incorporates a tool to validate information before trading
- User Authentication: Provide secure user authentication (login/signup) functionality.
- Theme Switching: Enable users to toggle between light and dark themes and persist this preference.

## Style Guidelines:

- Primary color: #2563eb (Strong blue), aligning with the request for a professional and reliable feel, without directly associating with cliches.
- Background: #f8fafc (Very light gray) for the light theme and #0f172a (Dark navy) for the dark theme as requested.
- Accent color: #1d4ed8 (A slightly darker, desaturated blue) for interactive elements and highlights, ensuring good contrast with the primary and background colors.
- Font: 'Inter' (sans-serif) for both body and headings as requested.
- Lucide-react icons should be consistently used throughout the application, in a size and weight that is visually balanced against the typography.
- Responsive 12-column grid layout with a fixed Navbar and sticky Footer, adapting to different screen sizes.
- Framer Motion for smooth page transitions and subtle widget animations.
